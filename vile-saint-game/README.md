# Vile Saint / 恶相圣人

一个中英文双语的网页小游戏：外表邪恶、内心善良的“怪物圣徒”在夜里偷偷保护村庄。

这版定位是高端暗黑童话：黑金/月光视觉、可玩 5 夜关卡、恐惧/善意/信任/诅咒数值、影幕潜行机制和多结局。

## 运行方式

直接打开 `index.html` 即可游玩。  
也可以把本文件夹里的所有文件上传到 `vilesaint.com` 的网站根目录。

## 操作

- 方向键 / WASD：移动
- 空格：互动 / 帮助
- E：使用影幕
- 站在阴影格上时，村民看不见你
- 完成每一夜的善事后，走到月门进入下一关
